◆ YumCTable

typedef struct YumCTable YumCTable

Opaque handle representing a Yum Table object.

Alias for YumEngine::YumTable (used by the C API).

Internally wraps a YumEngine::YumTable instance.

Warning
Objects of this type allocated via the C API must be released with YumCTable_delete() to avoid memory leaks.
