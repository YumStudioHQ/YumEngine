Detailed Description

Lua Subsystem management — integrates YumEngine's Lua runtime with native code.

This header defines the core classes responsible for managing Lua states and the binding of native callbacks to the Lua environment.
It provides two main components:

  • LuaSubsystem: manages a single Lua state and registered native callbacks.
  • Subsystem: manages multiple LuaSubsystem instances, identified by unique IDs.
