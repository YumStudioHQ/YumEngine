Detailed Description

C API for Yum Variant — dynamic typed value container.

This header provides a C wrapper around Yumcxx::Variant, which is a dynamically typed container used to store integers, floating-point values, booleans, strings, or raw binary data.

Variants are used extensively throughout the Yum API to pass values between Lua, native code, and user callbacks.
