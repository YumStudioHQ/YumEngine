Detailed Description

Minimal, type-safe container for Variant values.

YumEngine::Vector serves as a simplified wrapper over std::vector<Variant>. It is primarily used for passing arguments and return values between Lua scripts, native C++ code, and the C API.

The class is intentionally limited in scope to make cross-language interop (especially with C) simpler and predictable.

Example usage:

YumEngine::Vector args;
args.append(YumEngine::Variant(42));
args.append(YumEngine::Variant("hello"));
for (const auto& v : args)
std::cout << v.as_string() << std::endl;
